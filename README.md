# Parcelv2 React Starter

ReactJS stater boilerplate for parcel version 2

# Getting Started

## Clone this project

```
git clone https://github.com/fransbell/parcel2-react-starter.git
cd ./parcel2-react-starter
npm install
```

## Usage

Starting dev server

```
npm run dev
```

## Build Project

```
npm run build
```
